🎥 Video Chat App

A real-time video chat platform that lets users connect instantly with others through video, audio, and text.
Built using modern web technologies, the app enables seamless peer-to-peer communication directly from the browser.



🚀 Features

🔹 Real-time video & audio communication
🔹 One-to-one video calling
🔹 Instant connection with users
🔹 Integrated text chat
🔹 Mute / unmute audio & video controls
🔹 Simple and responsive UI
🔹 Browser-based (no installation required)

🧠 How It Works

1.Users open the app and join instantly
2.A signaling server connects peers using WebSockets
3.WebRTC establishes a direct peer-to-peer connection
4.Media streams (audio/video) are shared in real-time

🛠️ Tech Stack

☀ Frontend

☀ Backend

☀ Real-Time Communication
   WebRTC
   Socket.IO

☀ Deployment

🌍 Usage
1. Open the live app
2. Allow camera & microphone access
3. Start a video session
